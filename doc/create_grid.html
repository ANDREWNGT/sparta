create_grid command

Syntax: 

create_grid Nx Ny Nz
  • Nx,Ny,Nz = size of grid in each dimension

Examples: 

create_grid 10 10 10
create_grid 100 100 1

Description:

Overlay a regular Nx by Ny by Nz grid over the simulation domain defined by the create_box command.

For 2d simulations, Nz must be set to 1.

Restrictions:

This command must be used after the simulation box is defined by the create_box command.

Related commands:

create_box

Default: none
