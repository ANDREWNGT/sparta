write_grid command

Syntax: 

write_grid file
  • file = name of file to write grid info to

Examples: 

write_grid data.grid

Description:

Write a grid file in text format that lists all the parent cells in the currently defined hierarchical grid. This file is in the same format used by the read_grid command. Thus the file can be used to start a subsequent simulation using the same grid topology.

See the read_grid and create_grid commands for a definition of hierarchical grids and parent/child cells.

Restrictions: none

Related commands:

read_grid, create_grid

Default: none
