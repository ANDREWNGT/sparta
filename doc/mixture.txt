:line

background command :h3

[Syntax:]

background species nrho temp vx vy vz :pre

NOTE: This command is not yet implemented.

species = {all} or species ID
nrho = number density (# per length^3 units)
temp = thermal temperature (temperature units)
vx,vy,vz = streaming velocity (velocity units) :ul

[Examples:]

background all 1.0 300 0 0 0 :pre

[Description:]

Define properties of the background gas.

[Restrictions:] none

[Related commands:] none

[Default:] none
