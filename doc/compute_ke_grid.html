compute ke/grid command

Syntax: 

compute ID ke/grid
  • ID is documented in compute command
  • ke/grid = style name of this compute command

Examples: 

compute 1 ke/grid

Description:

Define a computation that calculates the per-atom translational kinetic energy of all the molecules in each grid cell.

The kinetic energy for the cell is simply the sum of 1/2 m v^2 over all the molecules in the cell, where m is the mass and v is the velocity of each molecule.

Output info:

This compute calculates a per-grid vector, which can be accessed by any command that uses per-grid values from a compute as input.

The per-grid vector values will be in energy units.

Restrictions: none

Related commands:

dump grid

Default: none
