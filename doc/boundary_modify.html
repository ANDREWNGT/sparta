boundary_modify command

Syntax: 

boundary_modify wall keyword value ...
  • wall = xlo or xhi or ylo or yhi or zlo or zhi
  • one or more keyword/value pairs may be listed 
    keywords = acc or temp
  acc value = frac
    frac = fractional value between 0.0 and 1.0 inclusive
  temp value = Twall
    Twall = temperature of wall (temperature units)

Examples: 

boundary_modify yhi acc 0.5 temp 300.0 

boundary_modify zlo acc 0.75

Description:

Set parameters for one of the boundaries of the global simulation box.

Any of the 6 faces can be selected via the wall setting.

The acc and temp keywords only apply to diffuse-style walls, as specified by the boundary command.

The acc keyword sets the accomodation coefficient. The temp keyword sets the wall temperature. See the boundary command doc page for details of how these parameters are used to reset a particle's attributes when it collides with a diffuse boundary.

Restrictions: none

Related commands:

boundary

Default:

The default values for all 6 walls are acc = 1.0 and temp = 273.15.
